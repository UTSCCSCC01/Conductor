---

This is a UI documentation page of **Orchestra**. 
Stories are break down into two parts - pages and components.

* **Pages** story contains the documentation of the Orchestra's pages.

* **Components** story contains the documentation of components that were used to build the Orchestra's pages.

Browse stories by navigating to them in the sidebar.
**Pages** story is more compatible with **Docs** than **Canvas**.
